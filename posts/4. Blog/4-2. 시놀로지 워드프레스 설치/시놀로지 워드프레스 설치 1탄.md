시놀로지 나스(Synology NAS)에 워드프레스를 설치하기 전에 기본적인 셋팅을 몇가지 해야합니다.

## 시놀로지 나스 워드프레스 설치 준비

워드프레스를 운영하면 외부로부터 내부(집 또는 사무실) 네트워크 환경에 유입이 생기기 때문에 보안을 강화해야하고, 설치 중 터미널 사용을 해야해서 SSH 서비스 사용 설정 그리고 워드프레스를 설치 할 공유 폴더를 생성해야합니다.

### 보안 설정 강화하기

시놀로지 나스를 사용하다보면 알 수 없는 외부의 누군가로부터 해킹 시도가 발생하곤 합니다. 그래서 방화벽 설정을 통해서 특정 국가나 IP 등 조건에 해당하는 접속을 차단하여 사용 중이실겁니다. 워드프레스를 설치하여 운영하는 것은 궁극적으로 외부 네트워크에 내가 운영하는 웹페이지를 노출하는 행위이며 내부 네트워크(집 또는 사무실)로 접속이 이루어진다는 이야기입니다. 그래서 보안에 신경을 써야하는데, 시놀로지 나스에서는 ‘보안 어드바이저’라는 기능을 통해 보안을 강화하기 위한 조치들을 안내해줍니다.

DSM 바탕화면에서 왼쪽 상단에 있는 ‘메인 메뉴’로 이동합니다.

![시놀로지-나스-DSM-메인-메뉴](https://blog.laribbongs.net/wp-content/uploads/2022/02/%EB%A9%94%EC%9D%B8-%EB%A9%94%EB%89%B4.jpg)

DSM 메인 메뉴

메인 메뉴에 있는 ‘보안 어드바이저’를 실행합니다.

![시놀로지-나스-보안-어드바이저](https://blog.laribbongs.net/wp-content/uploads/2022/02/%EB%B3%B4%EC%95%88-%EC%96%B4%EB%93%9C%EB%B0%94%EC%9D%B4%EC%A0%80.jpg)

메인 메뉴 내 보안 어드바이저 실행

보안 어드바이저의 고급 설정 메뉴에서 ‘직장 및 비즈니스의 경우’를 선택 후 저장합니다. 아무래도 가정 및 개인용보다 더 강화된 조건으로 현재 시놀로지 나스의 보안 상태를 점검하게 됩니다.

보안 어드바이저 고급 설정

고급 설정 후 왼쪽 탭에 개요를 눌러서 ‘스캔’을 실행하여 현재 나스의 보안 상태를 점검합니다.

### SSH 서비스 활성화

보안 설정이 완료 되었다면 터미널로 시놀로지 나스에 접속할 수 있도록 SSH 서비스를 활성화합니다. 시놀로지 나스 DSM의 제어판에서 ‘터미널 및 SNMP’ 메뉴로 들어가서 ‘SSH 서비스 활성화’를 체크하고 적용합니다. 포트번호는 임의로 설정해주셔도 되며 나중에 터미널로 접속할 때 필요합니다. 일반적으로 SSH 서비스 포트 번호는 22번입니다.

![시놀로지-나스-SSH-서비스-활성화](https://blog.laribbongs.net/wp-content/uploads/2022/02/SSH-%EC%84%9C%EB%B9%84%EC%8A%A4-%ED%99%9C%EC%84%B1%ED%99%94-1024x512.jpg)

SSH 서비스 활성화 적용

### 공유 폴더 생성하기

시놀로지 나스 워드프레스 설치 준비 마지막 단계로 워드프레스를 설치할 폴더를 생성하겠습니다. DSM 제어판에 있는 공유 폴더 메뉴로 들어가서 새로운 공유 폴더 생성을 합니다.

![시놀로지-나스-신규-공유-폴더-생성](https://blog.laribbongs.net/wp-content/uploads/2022/02/%EA%B3%B5%EC%9C%A0-%ED%8F%B4%EB%8D%94-%EC%83%9D%EC%84%B1.jpg)

새로운 공유 폴더 생성

폴더명과 폴더를 생성할 위치를 지정해줍니다. 저는 폴더명을 testwordpress로 정하고 볼륨2에 생성하도록 하겠습니다. 참고로 저는 시놀로지 나스의 저장공간을 볼륨1, 볼륨2로 구분하여 사용중입니다. 볼륨1에는 개인적인 자료들을 모아두고 볼륨2는 웹서비스 전용으로만 사용 중입니다. 혹여나 해킹 피해를 예방할 수 있을 것 같다는 판단으로 이렇게 지정하여 사용합니다.

![공유-폴더-기본-정보-설정](https://blog.laribbongs.net/wp-content/uploads/2022/02/%EA%B3%B5%EC%9C%A0-%ED%8F%B4%EB%8D%94-%EC%83%9D%EC%84%B1-%EA%B8%B0%EB%B3%B8-%EC%84%A4%EC%A0%95.jpg)

공유 폴더 고급 설정

마지막으로 폴더에 대한 사용자 권한을 지정합니다. 나스에 등록된 사용자 계정 중에서 새롭게 생성한 폴더에 접근할 수 있는 사람에게 읽기/쓰기 권한을 주면 됩니다. 사용자 권한까지 설정하면 워드프레스 설치를 위한 폴더 생성이 끝이 납니다.
